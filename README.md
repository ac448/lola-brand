# LOLA

LOLA is an AI project manager for creative teams. She reads incoming work requests and routes them on her own. Any call can be undone by a human in one click. This repository holds the static brand site for the project. It contains the case study page plus spec sheets for the status chip system and the request architecture. Everything is plain HTML and CSS with no build step, served from GitHub Pages. LOLA was built and executed through Technovation.
