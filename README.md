# LOLA

LOLA is an AI project manager for creative teams. She reads incoming work requests on the team's Asana board and routes them to an owner. Today she runs in safety mode, proposing every call for a human to approve. A gated live mode, where she assigns work on her own, is the next step. This repository holds the static brand site for the project. It contains the case study page plus spec sheets for the status chip system and the request architecture. Everything is plain HTML and CSS with no build step, served from GitHub Pages. LOLA was built and executed through Technovation.
